# Incoterms 2020 - Complete Reference

> Source: Research Agent 1 (2026-02-22)
> Covers: All 11 rules, obligations, risk/cost transfer, Indonesian examples

## Key Changes: Incoterms 2010 to 2020

| Change | Detail |
|--------|--------|
| DAT renamed to DPU | "Delivered at Place Unloaded" - destination can be ANY place, not just terminal |
| CIP insurance upgraded | CIP now requires ICC Clause A (all-risk). CIF remains at Clause C (basic) |
| FCA B/L option | Buyer can instruct carrier to issue on-board B/L to seller (solves L/C problem) |
| Own transport allowed | FCA, DAP, DPU, DDP allow seller/buyer own transport |
| Security requirements | Transport security obligations added to A4 and A7 |
| Cost consolidation | All costs now appear in A9/B9 for clarity |

## Insurance Clauses Quick Reference

| Clause | Level | Used By | Covers |
|--------|-------|---------|--------|
| ICC Clause C | Basic | CIF | Fire, explosion, sinking, collision, jettison, general average |
| ICC Clause B | Medium | By agreement | Clause C + earthquake, lightning, washing overboard, sea water entry |
| ICC Clause A | All-Risk | CIP | All risks EXCEPT war, strikes, inherent vice, delay |

## GROUP E - DEPARTURE

### EXW (Ex Works)
- **Transport:** Any mode
- **Risk transfer:** Goods placed at buyer's disposal at named place (NOT loaded)
- **Seller:** Packaging only. No loading, no export clearance, no transport
- **Buyer:** Everything from collection onward including export clearance
- **Common mistake:** Buyer must handle EXPORT from seller's country (impractical for international trade)
- **ICC recommendation:** Use FCA instead for international sales
- **Indonesia example:** PT Furniture Jaya Jepara sells teak to Australian buyer. Buyer's agent collects from workshop, arranges trucking to Semarang/Tanjung Emas, handles Indonesian export customs

### FCA (Free Carrier)
- **Transport:** Any mode (recommended replacement for FOB with containers)
- **Risk transfer:** At seller's premises = loaded on buyer's vehicle. Elsewhere = on seller's transport, ready for unloading
- **Seller:** Delivery to carrier, export clearance, local transport
- **Buyer:** Main carriage, insurance, import clearance
- **2020 B/L option:** Buyer instructs carrier to issue on-board B/L to seller for L/C
- **Key advantage:** Correct term for containerized goods (NOT FOB)
- **Indonesia example:** PT Spice Indo Surabaya sells spices via Tanjung Perak. Seller delivers container to terminal, risk transfers to carrier

### FAS (Free Alongside Ship)
- **Transport:** Sea/waterway ONLY
- **Risk transfer:** When goods placed alongside vessel at named port
- **Seller:** Delivery alongside ship, export clearance
- **Buyer:** Loading onto vessel, ocean freight, everything after
- **For:** Breakbulk/bulk cargo only (NOT containers)
- **Indonesia example:** PT Kayu Nusantara sells timber logs at Banjarmasin port. Logs placed on quay alongside vessel

### FOB (Free On Board)
- **Transport:** Sea/waterway ONLY
- **Risk transfer:** When goods are on board the vessel
- **Seller:** Delivery on board, loading, export clearance
- **Buyer:** Ocean freight, insurance, import clearance
- **#1 misuse globally:** Using FOB for containers. Containers go to terminal BEFORE vessel loading - gap between control and risk transfer. Use FCA instead
- **Indonesia example:** PT Kelapa Sawit Dumai exports CPO in bulk tanker. Risk transfers when oil pumped on board

## GROUP C - MAIN CARRIAGE PAID

### CFR (Cost and Freight)
- **Transport:** Sea/waterway ONLY
- **Risk transfer:** On board vessel at port of shipment (same as FOB)
- **Cost transfer:** Seller pays freight to destination port
- **Key trap:** Risk and cost split at different points! Risk = origin, Cost = destination
- **Seller:** Delivery on board + ocean freight + export clearance
- **Buyer:** Insurance (highly recommended), unloading, import clearance

### CIF (Cost, Insurance and Freight)
- **Transport:** Sea/waterway ONLY
- **Risk transfer:** On board vessel at port of shipment
- **Insurance:** Seller MUST buy minimum ICC Clause C (basic only!)
- **Common misunderstanding:** CIF insurance is MINIMUM coverage, not comprehensive
- **Seller:** Delivery on board + ocean freight + basic insurance + export clearance
- **Buyer:** Unloading, import clearance, may want additional insurance

### CPT (Carriage Paid To)
- **Transport:** Any mode (multimodal version of CFR)
- **Risk transfer:** Delivery to first carrier
- **Cost transfer:** Seller pays carriage to named destination
- **Seller:** Delivery to carrier + main carriage + export clearance
- **Buyer:** Insurance, unloading, import clearance

### CIP (Carriage and Insurance Paid To)
- **Transport:** Any mode (multimodal version of CIF)
- **Risk transfer:** Delivery to first carrier
- **Insurance:** Seller MUST buy ICC Clause A (ALL-RISK) - higher than CIF!
- **Key difference from CIF:** CIP = Clause A (all-risk), CIF = Clause C (basic)
- **Seller:** Delivery to carrier + main carriage + all-risk insurance + export clearance
- **Buyer:** Unloading, import clearance

## GROUP D - ARRIVAL

### DAP (Delivered at Place)
- **Transport:** Any mode
- **Risk transfer:** At named destination, ready for unloading
- **Seller:** All transport + insurance to destination, export clearance
- **Buyer:** Unloading + import clearance + duties + taxes
- **Indonesia example:** Chinese machinery seller delivers DAP buyer's factory in Cikarang. Buyer handles Indonesian customs

### DPU (Delivered at Place Unloaded)
- **Transport:** Any mode
- **Risk transfer:** At named destination, AFTER unloading
- **Only Incoterm where seller unloads at destination**
- **Renamed from DAT (2010) to DPU (2020)** - can be any place, not just terminal
- **Seller:** All transport + insurance + unloading at destination + export clearance
- **Buyer:** Import clearance + duties + taxes

### DDP (Delivered Duty Paid)
- **Transport:** Any mode
- **Maximum seller obligation** - opposite of EXW
- **Risk transfer:** At named destination, ready for unloading
- **Seller:** Everything including import clearance, duties, taxes, VAT
- **Buyer:** Unloading only
- **Challenge:** Seller must be registered for import in buyer's country
- **Indonesia context:** Foreign seller must have NIK/API or use PPJK agent for Indonesian import clearance
