# Belajar Ekspor Impor

Interactive Incoterms & Export-Import Simulator Game. Learn Indonesian freight forwarding through realistic shipping scenarios.

## Features

- **Learn Incoterms 2020** - All 11 rules with visual guides, obligations, and Indonesian trade context
- **Scenario Simulator** - Pick the correct Incoterm for real-world shipping scenarios from/to Indonesia
- **Cost Simulator** - Compare total costs (seller vs buyer) across different Incoterms
- **Progress Tracking** - Track mastery per Incoterm, accuracy, streaks, and scores

## Tech Stack

| Component | Technology |
|-----------|-----------|
| Framework | React 19 + Vite |
| Styling | Tailwind CSS v4 |
| State | Zustand (persisted to localStorage) |
| Animation | Framer Motion |
| Icons | Lucide React |
| Deploy | Vercel |
